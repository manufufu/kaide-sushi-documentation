How to use Kaide sushi

How to select the right adds-on for the given product

Radio button

  1. Select the product that you want to be adds-on, and add the product type of

    "adds-on--radio" as product type.

  2. Create a collection and add products you want to output as adds-on

  3. In the navigation, use your created collection as a dropdown menu.

    It is important to know that the EMPTY collections in there means that there will be no adds-on for that product

  4. Why there's empty collection? it is there to target the product you want to have the adds-on.

Checkbox

  1. Select the product that you want to be adds-on, and add the product type of

    "adds-on--cb" as product type.

  2. Create a collection and add products you want to output as adds-on

  3. In the navigation, use your created collection as a dropdown menu.

  4. Output

Variant selector

  1. It is important to have to have a variant for the selected product, only the first variant will only be selected

  2. Create a dummy product which will serve as a variant option for that product.

  3. Create a collection for the dummy product that you created.

  4. Use the created collection as a navigation dropdown.

  5. Output

How to output "/each" or "from" for specific products.

Output "/each" from specific product.

  1. Select the product that you want to have the /each

  2. Add "eachItem" in the product descrition.

    adding this wil let the theme determine that the product will have /each

  3. Output

Output "from" for specifc product.

  1. Select the product that you want to have the "from"

  2. Add "eachItem" in the product descrition.

    adding this wil let the theme determine that the product will have "from"

  3. Output

How to remove label from specific product.

  1. Select the product.

  2. Add "removeLabel" to product description, the theme will now know what to do with it.

  3. Output.

    No label

    With label
